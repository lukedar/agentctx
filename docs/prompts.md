# Prompts

These are copy/paste prompts designed for Copilot and other coding agents.

## Builder prompt
Use `createBuilderPrompt(task)` to generate the Builder role prompt.

## Evaluator prompt
Use `createEvaluatorPrompt(task)` to generate the Evaluator role prompt.

## Full “dual-agent run” wrapper
Use `createCopilotDualAgentRunPrompt(task)` to generate the end-to-end process wrapper.

Tip: keep tasks small and force real Decision Records. That’s where quality improves.
